# IELTS Listening Question Generator (FYP)
## Introduction
This is my final year project titled "IELTS Question Generator for Listening Module by Using LLM (Gemini)"
The sysetm leverages large language models to automatically generate IELTS-style listening questions along with corresponding audio files, helping students practice more efficiently.

## Features
* Generate IELTS listening questions automatically with audio.
* Download questions and audios in PDF and Text format.
* Customize themes, question types and other specifications.
* User-friendly web interface for easy interaction.

## Technologeis Used
* Backend: Python, Flask
* Frontend: HTML, CSS, JavaScript
* Database: Local files & Google Firestore
* AI Model: Gemini LLM

## Training Set Resources
* https://engexam.info/ielts-listening-practice-tests-printable/

## Installation and Setup
1. Make sure you are in the folder that contains app.py
2. Open terminal and run the following commands:
pip install -r requirements.txt
python app.py
3. Open your browswer and navigate to http://127.0.0.1:5000 to access the application

## Usage
1. Input your desired settings and preferences for the listening questions.
2. Click Generate to produce questions and audios.
3. Preview the generated questions and audio.
4. Download the results in PDF or Text format.
